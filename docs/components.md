
## Modal

Modal components

```html
<Modal
  show={this.state.visiable} close={this.setInvisiable} />
```


### 选项

| parameter   | type     | default value | description                                         |
| ----------- | -------- | ------------- | --------------------------------------------------- |
| show     | Boolean    |               | required. Determine if the modal is displayed 
| close    | Function   |               | fuction set modal invisiable
